Course Hero Bot

Summary

CH Bot is a discord bot that has been in over +500 discord servers and has helped over +10000 students around the world access millions of study resources on Course Hero. Built with Python, CH Bot allows for a simple, yet effective way for user to generate unlocks free of charge.

Bot Explanation

In order to unlock study documents on Course Hero, you need unlocks. Course Hero allows users to upload study documents on their website for unlocks; for every 10 documents you upload, you will recieve 5 unlocks. What CH bot does is it generates the amount of documents you want and you upload them to Course Hero for free unlocks!

Fun Facts

  • 100% developed by me
  • Technologies Used:
    • Python
    • Discord API

  • +3 million documents generated! ➜ 3m/10 = 300k unlocks recieved!

How to Use

WARNING: Do not upload more than 40 documents per day, it will get auto rejected

  1. Do !f 20 in the #ch-bot channel in discord and download the zip file
  2. Go to https://www.coursehero.com/upload
  3. Make a account if you didn't already
  4. Extract/open the .zip file and upload the documents
  5. Select random class for the documents
  6. Wait for documents to approve!
